# gbank-python
## 檔案結構
```
gbank/
    ├── app.py
    ├── database.py
    ├── templates/
    |   ├── index.html
    │   ├── base.html
    │   ├── admin.html
    │   └── user.html
    └── static/
        └── style.css
```
## 如何使用
1. 安裝PIP
`pip install flask pytz`
2. start db
`python database.py`
3. start flask
`python app.py`
## default user
default admin is root
